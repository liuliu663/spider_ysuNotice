关于举办“微软高性能计算校园编程大赛”的通知
2011-04-26 08:15     (点击： )
一、活动主旨
        为了普及新技术、新知识，推广高性能计算的应用，微软（中国）有限公司决定将在中国高校内举办“微软高性能计算校园编程大赛”。竞赛旨在通过微软高性能计算技术，使工程、生物技术、医药、地质和计算等领域的计算研究不再受制于昂贵的基础结构和难于发现的资源。 
        本次竞赛将给参赛的院系最大的自由度去选择合适自己的基于Windows HPC Server的项目参赛. 使用Windows HPC Server在院系自己的研究项目，课题当中，无论是开放源代码移植，并行算法研究，串行程序并行化，计算设计过程优化，甚至是基于多机环境的批处理作业，一切的一切，释放您的潜力与创造力。
二、竞赛组织机构 
        主办单位: 微软（中国）有限公司（以下简称“微软”） 
        协办单位: 中国教育网www.edu.cn 
        中国教育在线www.eol.cn 
        《中国教育网络》杂志
三、竞赛时间 
        2011年3月28日至2011年6月30日
四、竞赛参与区域及人员 
        活动对象：所有中国大陆地区的高等院校 
        参赛人员包括在参赛区域内任何民办或公立学院/大学（以下简称“参赛机构”）中在册的所有本科生和研究生（包括外国学生，以下简称“参赛者”），都有资格参加此次竞赛。但微软公司及其隶属机构、子公司、及协办单位(中国教育网www.edu.cn、中国教育在线www.eol.cn、《中国教育网络》杂志)的雇员不能参与此次竞赛，这些职员的家庭成员也不得参赛（家庭成员包括父母、子女、兄弟姐妹、配偶和生活伴侣）。
五、竞赛者参与办法 
适用产品：Windows HPC Server 2008 R2 
试用许可：Academic
参与方式： 
        1）参赛者需组成一个项目团队（以下简称“项目团队”），团队中至少有一（1）名参赛者，最多有三（3）名参赛者。项目团队应由参赛机构中为参赛项目描述论文提供帮助的研究指导教师（以下简称“指导教师”）监督和指导。 
        2）为了参与此次竞赛和开发竞赛项目提案，参赛者需要拥有一套多机环境的计算机系统。这个系统必须在基于 x64 CPU (Intel 或 AMD) 的硬件平台上运行 Windows HPC Server 2008 R2。所支持的应用程序和系统要求的详细内容，请访问 http://www.microsoft.com/china/hpc/ 
        3）参赛机构中参加竞赛的项目团队数量不限。 
        4）参赛者在在线注册时，需同时提供一份不少于50字的参赛团队及指导老师背景基本介绍，以供初步审核。在得到参赛资格后，必须在2011年5月31日前提交一份描述Windows HPC Server 2008 R2创新使用的项目详细描述论文（“参赛项目”） 
        5）参赛项目的研究方向包括但不只限于工程、生物技术、医药、地质和计算等领域等。
六、竞赛参与形式 
        竞赛报名和参赛项目提交均采用网络方式。每个项目团队都必须在网站<http://mshpc.eol.cn> 进行注册，才可以参与此次竞赛。在注册时，必须提供参赛者、指导教师和参赛项目的详细信息。注册同时需提供团队基本介绍，在初步审核通过后，参赛团队会得到大赛官方的正式通知邮件。 
        参赛项目提交要求WORD文档格式，不需要书面。
七、参赛者优惠 
        在2011年6月30日以前，购买正版Windows HPC Server 2008 R2教育版，更可享有优惠，详情请与当地微软经销商联系。
八、参赛者选拔 
        1、此次比赛的获奖人员将在 2011 年 6月30日公布。 
        2、赢得此次竞赛的项目团队中的参赛者（“获奖学生”）将获得6, 000元（陆仟元整）的人民币奖励（“学生奖金”）。根据项目团队中的参赛者人数，这笔奖金最多由三（3）名参赛者分享。 
        3、负责督导获奖项目团队的指导教师（“获奖指导教师”）将获得30, 000元（叁万元整）的人民币奖励，作为教师研究补助金（“指导教师奖金”）。同一指导教师指导的多篇获奖作品，可累计计算。 
        4、学生奖金将颁发给参与前十（10）名项目建议书的参赛者，并从中评选出特等奖（至多3名）而指导教师奖金将颁发给负责督导特等奖项目建议书的指导教师。获奖人员将由微软从学术界和行业中选出的专家小组评选。微软就有关获奖学生和获奖指导教师所做的决定为本次大赛具有约束力的最终决定。 
        如有无法选出符合标准的参赛项目的情况，评委会保留不颁发奖项的权利。评分点：课题创新度（20分）、课题价值度（25分）、课题的整体设计和现实性（20分）、用户体验感（25分）、参赛文交付件的完整性（10分）。前10名作品按照得分高低顺序排出；特等奖作品须达到85分以上，如有多于3篇作品满足获奖条件，则按照分值高低优先选取。
5、奖励条件： 
        学生奖金和指导教师奖金不可转让，也不能兑换成现金或其他东西。微软特此保留用同等价值或更高价值的奖励取代这些奖金的权利。 
        微软保留取消任何违反竞赛条约的获奖者奖励的权利。 
        由任何有关政府或税收机构对学生奖金和指导教师奖金征收的税款完全由获奖学生和获奖指导教师承担，微软（中国）有限公司将承担代扣代缴义务，并直接从奖金中将个人所得税款扣除缴纳主管税务局。
九、竞赛时间安排如下 
        注册提交时间：2011年3月28日－2011年5月31日（以网站注册时间为准） 
        复审时间：2011年6月1日—6月3日 
        终审时间：2011年6月4日—6月8日 
        大赛官方网站公布大赛最终获奖作品：2001年6月16日—6月25日 
        大赛颁奖时间：2011年6月30日前
