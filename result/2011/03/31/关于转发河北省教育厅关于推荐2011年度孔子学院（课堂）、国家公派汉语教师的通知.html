关于转发河北省教育厅关于推荐2011年度孔子学院（课堂）、国家公派汉语教师的通知
2011-03-31 15:26     (点击： )

校属各单位：
      根据国家汉办《关于推荐2011年孔子学院（课堂）、国家公派汉语教师的函》（汉办[2011]147号）和河北省教育厅《关于推荐2011年度孔子学院（课堂）、国家公派教师的通知》的要求，我校现开展推荐2011年度美国托力多大学孔子学院汉语教师及国家公派汉语教师人选工作，有关事项通知如下：
一、 选拔对象：我校在职教师，但不包括以下人员：
      1. 学校现任副处级（含）以上干部；
      2. 回校（国）工作不满5年者；
      3. 已经申报国家或地方各类公派留学项目者；
      4. 参加过国家汉办公派汉语教师或孔子学院教师选拔未赴任者；
二、 选拔条件
（一）国家公派汉语教师
      1. 热爱汉语国际推广事业，具有较强的使命感、光荣感和责任感。师德高尚，爱岗敬业，组织纪律性强，有团队精神。
      2. 具有2年以上教龄的学校在职在编教师（包括聘用的合同制教师）；大学本科以上学历，对外汉语、汉语言文学、外语、教育、心理、历史、文化等人文学科专业学习背景。有1年以上汉语作为第二语言教学经验、国外学习工作经历者优先；具有较强的汉语教学、中华文化传播和跨文化交际能力。
      3. 汉语普通话水平测试须达到二级甲等以上（含）。至少熟练掌握一门外语。
      4. 原则上，年龄在50周岁（含）以下，非英语语种的教师年龄可放宽至55周岁。
      5. 具有良好的身体素质，身体健康，有较强的适应能力和心理承受能力。
      6. 符合教师岗位具体要求。
（二）美国托列多大学孔子学院汉语教师
      除满足以上条件外，还需满足《孔子学院教师任职条件》：
      1. 遵守《孔子学院章程》和总部有关规定。
      2. 遵守孔子学院所在国有关法律，尊重当地风俗习惯，努力树立孔子学院教师的良好形象。
      3. 遵守孔子学院有关管理规定，接受孔子学院院长、副院长和学校的双重领导。
      4. 热爱本职工作，积极承担孔子学院教学工作，按时完成教学任务。
      5. 按照孔子学院要求，积极参与孔子学院开展的汉语言文化推广活动及其它相关工作。
      6. 任职资格条件：
      （1）大学本科（含）以上学历。 
      （2）一般应具有讲师（含）以上职称。 
      （3）语言教师普通话要达到国家二级甲等或相当于二级甲等的水平。 
      （4）具备英语语言知识及技能。 
      （5）具有汉语作为外语教学、课程设置、教学评估、教师培训等方面的能力和经验。 
      （6）具有熟练运用现代信息技术并服务于教学的能力。 
      （7）了解和掌握中国文化基本知识和当代中国概况，具有良好的文化活动策划与组织能力。 
      （8）身心健康，诚实、正直，性格开朗，具有良好的沟通和交往能力。 
      （9）爱岗敬业，具有团队精神。
三、推荐程序 
      1. 提交材料 
      （1）《燕山大学校内申请表》（附件1） 
      （2）申请人须登陆http://shizi.chinese.cn网址，自愿选择孔子学院（课堂）教师或者国家公派汉语教师填写申请，不可双重申请。填妥《孔子学院（课堂）汉语教师申请表》或《国家公派汉语教师申请表》后直接网上提交，同时下载纸质表格提交国际教育学院审核。不在网上填写的不予受理。 
      （3）填写《河北省2011年孔子学院（课堂）、国家公派教师申报人员信息统计表》（附件2） 
      （4）各种辅助证明材料复印件，包括：最高学历证书、普通话等级证书、个人身份证件、外语水平证书等。 
      2. 学校审核 
      学校根据申请人所提交材料进行审核，并向国家汉办/孔子学院总部推荐。 
      3. 汉办审核 
      孔子学院汉语教师和国家公派教师推荐人选须参加国家汉办统一组织的面试选拔，具体安排另行通知。
四、培训 
      面试后择优录取。录取的教师须接受国家汉办组织的相关培训，培训时间拟定于7月，期限2个月，培训考核合格后派出。培训具体安排另行通知。
五、派出及待遇 
      培训合格人员按综合条件排名，根据排名先后决定派出顺序。 
      教师派出前须与学校签署《在职培训进修协议》； 
      教师待遇按照《国家公派出国教师生活待遇管理规定》执行。
六、报名截止日期 
      请有意申报者于4月2日上午12点前将申请材料报送至国际教育学院1715办公室。
