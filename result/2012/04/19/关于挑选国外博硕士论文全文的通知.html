关于挑选国外博硕士论文全文的通知
2012-04-19 10:59     (点击： )

    ProQuest Dissertations & Theses（PQDT）是世界著名的学位论文数据库，收录有欧美2,000余所大学270多万篇学位论文的文摘信息，涵盖文、理、工、农、医等各个学科领域，是迄今为止世界上最大的国际性博硕士论文数据库。ProQuest学位论文全文数据库收录的是PQDT数据库中部分记录的全文。
    从2002年起，CALIS开始组织ProQuest学位论文全文数据库CALIS集团采购（简称“CALIS集团”）。这个集团的核心是，由每个参与的成员馆购买一部分学位论文全文，集团内所有的学位论文放在服务器上共享，各个学校的校园网用户可免费下载这些学位论文。截止目前，CALIS集团已经进行了三个合同期。为了促进学校科研水平的提高，我校图书馆已于去年起加入CALIS集团采购，并由读者挑选了200篇学位论文，现已开通使用。按照采购方案，我校今年仍将挑选200篇学位论文作为永久馆藏，并可以共享使用其他170余所高校本年度订购的全文。
    注：因可选论文数量有限，本次仅限教师和博士研究生选择，请提交的读者尽量按照少而精的原则进行选择，每人限选10篇。论文提交成功后可于2012年6月份在Proquest学位论文库中获取全文。

ProQuest学位论文全文选择方法
一、进入ProQuest学位论文文摘库
   登陆学位论文文摘库：http://proquest.umi.com/pqdweb    账户: 4J2TVJBRD9 密码: welcome 。
    缺省情况下，您会看到基本检索(Basic Search)页面，如果您需要进行高级检索，可以点击高级（Advanced）标签。
 
二、简单检索
1. 在简单检索的界面，您可以在检索框中输入检索词进行检索。也可以在“日期范围”这个按钮中选择灵活的时间组合，以便对关键词进行有效的时间限定。
2. 如果您想要在特定的字段进行检索，以提高检准率，请点击 “更多检索选项”，这样可以在特定的检索入口中进行检索。
3. 输入检索式后，点击“检索”开始检索。
4. 在“检索结果（Results）”界面中对检索结果浏览，同时对希望订阅的论文（请尽量挑选“摘要”、“预览”、“订购”三项俱全的论文）进行标记（marked）（点击前面的方框，显示“√”，同时反显）；（注意：论文号(AAI字段)以数字0和字母C、D、T、U开头的论文，ProQuest公司是无法提供全文的，因此请不要选择、提交此类型论文。）
5. 论文挑选完毕后，点击栏目“标注的项目（Marked List）”，显示挑选完毕的论文篇目列表；登录国内高校已购买的学位论文全文库（http://pqdt.calis.edu.cn/），按照题名检索选中的论文，如果能检索到，说明国内已有高校购买，我校将不再重复购买，请重新选择其他论文。
6. 确定您标记的论文没有问题后，点击“导出(Export)”将论文导出；
7. 选择论文导出格式：右键单击第三项“以与 ProCite、EndNote、Reference Manager 和 RefWorks 兼容的格式下载”（Download in a format compatible with ProCite, EndNote, Reference Manager and RefWorks），选择“目标另存为”，并保存为文本文件。如需浏览，请使用写字板打开文档。
8. 将下载下来的文本文件于5月31日之前发送到邮箱song@ysu.edu.cn，信中注明所在姓名、院系、身份（教师、博士生）、ID卡号，完成挑选论文的工作。

三、高级检索
1. 在“高级检索（Advanced）”界面，系统默认可以进行三个检索式（通过下面的“添加一行”可以最多达到七个检索式）的检索，这三个检索条件可以按布尔逻辑算符（包括AND、OR、AND NOT、WITHIN 3、PRE/1）进行组合。
2. 高级检索可以提供更多的检索入口。
3. 输入检索式后，点击“检索”开始检索。
4. 重复上文中简单检索的步骤4-8。
 
四、浏览
1. 在“浏览（Browse）”界面上，您可以“按学科（By Subject）”或者“按位置（By Location）”点击“查看文档（View documents）”来浏览论文，在页面的上方提供了A to Z导航。
2. 重复上文中简单检索的步骤4-8。

如在使用过程中遇到任何问题，请随时与图书馆电话联系：8063942，宋老师。
